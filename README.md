# AQI Predictor 

##  Project Overview

This is a beginner-friendly Machine Learning project that predicts the
Air Quality Index (AQI) using pollutant data such as PM2.5, PM10, and
NO2.

The project demonstrates basic steps of a data science workflow: - Data
loading - Data cleaning - Model training - Prediction - Evaluation -
Visualization

------------------------------------------------------------------------

## Technologies Used

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Scikit-learn
-   Joblib

------------------------------------------------------------------------

## Dataset

The dataset should be in CSV format and must include the following
columns: - PM2.5 - PM10 - NO2 - AQI

Make sure the file is named:

    aqi_data.csv

------------------------------------------------------------------------

## Features of the Project

-   Handles missing data automatically
-   Uses Linear Regression model
-   Shows prediction results
-   Displays model accuracy (R² score)
-   Visualizes actual vs predicted AQI
-   Takes user input for real-time prediction
-   Saves trained model as `.pkl` file

------------------------------------------------------------------------

## Model Details

-   Algorithm: Linear Regression
-   Input Features: PM2.5, PM10, NO2
-   Output: AQI

------------------------------------------------------------------------

## Output

-   Predicted AQI values
-   Accuracy score
-   Graph showing actual vs predicted AQI
